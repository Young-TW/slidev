# 安裝 Github-cli

在瀏覽器搜尋 Github cli 然後進到  

```
https://cli.github.com
```

直接下載  

如果是 linux 或 macOS 使用者也可以用套件管理工具安裝  

```sh
// apt
sudo apt install gh
// homebrew
brew install gh
```
